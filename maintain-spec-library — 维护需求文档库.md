# 维护需求文档库

## 触发条件
用户说"把这份需求加到文档库"或"找一下[功能]的需求文档"。

## 输入
- 需要归档的需求文档，或搜索查询
- （可选）标签、版本信息、状态

## 步骤
1. 如果是归档：按标题、日期、状态和标签索引需求
2. 如果是搜索：扫描文档库返回匹配的需求
3. 跟踪需求版本和状态（草稿/评审中/已批准）
4. 标记长时间未更新的过期需求

## 输出
有序的需求索引，支持搜索、版本管理和状态跟踪。
